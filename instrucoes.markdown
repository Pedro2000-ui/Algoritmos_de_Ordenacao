Para criar o .exe:
    - Estando na raiz, entre na pasta referente ao algoritmo de ordenação que quer executar. Exemplo: cd bubble-sort
    - rode o comando gcc -o main.exe main.c

Para executar o .exe
    - Estando na raiz, entre na pasta referente ao algoritmo de ordenação que quer executar. Exemplo: cd bubble-sort
    - rode o comando ./main.exe
